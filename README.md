# Online Retail Data Analysis – SQL & Power BI

## Project overview

This project analyzes online retail sales data using SQL and Power BI.

The goal was to clean raw transactional data, perform analysis using SQL, and create an interactive Power BI dashboard to visualize key business metrics.

## Project workflow

1. Data cleaning and analysis using SQL (PostgreSQL)
2. Data visualization using Power BI
3. Business insights and reporting

## Tools used

- SQL (PostgreSQL)
- Power BI
- Data visualization
- Data analysis

## Project files

- sql/online_retail_analysis.sql → data cleaning and analysis
- Online_Retail_Dashboard.pdf → dashboard preview
- REPORT.md → detailed analysis report

## Skills demonstrated

- SQL data cleaning and analysis
- Data transformation
- Power BI dashboard development
- Business intelligence and reporting
